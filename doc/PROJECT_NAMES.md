## 命名のベストプラクティス

### ✅ 良い名前の特徴
1. **短い**: 2-3単語以内
2. **覚えやすい**: 発音しやすい
3. **意味が明確**: 何をするツールか分かる
4. **ドメイン取得可能**: .com/.io/.devが取れる
5. **検索しやすい**: タイポしにくい

### ❌ 避けるべき名前
1. **長すぎる**: 覚えにくい
2. **タイポしやすい**: スペルミスされやすい
3. **既存サービスと被る**: 商標問題
4. **意味が不明確**: 何をするツールか分からない
